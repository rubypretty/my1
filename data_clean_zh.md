# data_clean_zh.py 清理紀錄

## 程式

- 程式檔名：`data_clean_zh.py`
- 輸入資料庫：`seventeen_clean.sqlite3`
- 輸入資料表：`posts`
- 輸出資料庫：`seventeen_clean_zh.sqlite3`
- 輸出資料表：`posts`

## 輸出欄位

| 欄位 | 說明 |
| --- | --- |
| `serial_id` | 新的流水編號，從 1 開始 |
| `num` | 原始 `seventeen.sqlite3` 的 `posts.num`，由 `seventeen_clean.sqlite3` 保留下來 |
| `main_text` | `seventeen_clean.sqlite3` 的 `posts.main_text` |

## 清理規則

1. 排除指定片語。
   - 只要 `main_text` 含有以下任一片語，就整筆刪除：
     - `AI 資訊`
     - `尚無回覆`
     - `查看動態`

2. 判斷中文比例前，先忽略 `翻譯`。
   - 程式會先把 `main_text` 中的 `翻譯` 暫時移除，再判斷是否為中文資料。
   - 這是為了避免只有 `翻譯` 兩個中文字的外文資料被誤判為中文。
   - 輸出的 `main_text` 不會刪掉原文中的 `翻譯`，只是判斷時忽略。

3. 用中文字比例判斷是否保留。
   - 中文漢字數至少 `2` 個。
   - 中文漢字數 / 有意義文字數 >= `30%`。
   - 日文假名數 / 有意義文字數 <= `20%`。
   - 有意義文字指 `isalpha()` 或 `isdigit()` 判定為文字或數字的字元。

4. 保留通過條件的資料。
   - 通過中文比例條件的資料會重新編 `serial_id`。
   - `num` 和 `main_text` 保留原資料內容。
